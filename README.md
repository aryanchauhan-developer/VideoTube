# VideoTube

A YouTube-inspired video sharing platform built using the MERN stack.

## Features

### Authentication & User Management

* User Registration
* User Login & Logout
* JWT Authentication
* Access Token & Refresh Token
* User Profile Management
* Avatar Upload
* Cover Image Upload

### Video Management

* Upload Videos
* View Videos
* Update Video Details
* Delete Videos

### Social Features

* Like Videos
* Comment on Videos
* Subscribe to Channels
* View User Profiles

## Tech Stack

### Backend

* Node.js
* Express.js
* MongoDB
* Mongoose
* JWT
* Multer
* Cloudinary
* Bcrypt

### Frontend

* React.js
* React Router
* Axios
* Context API

## Installation

### Backend

```bash
cd backend
npm install
npm run dev
```

### Frontend

```bash
cd frontend

npm run dev
```
